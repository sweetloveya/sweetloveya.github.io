简介

Git Bash是一个Microsoft Windows应用程序，具有Git命令行shell体验和实用程序，例如安全外壳协议(SSH)、安全复制协议(SCP)、CAT（连接命令）等。

Bash是Bourne Again Shell 的首字母缩写词，这是一个 GNU 项目 shell。shell用于通过执行命令与操作系统交互，Bash是 Linux 和 macOS 上使用的默认 shell。

命令

cd :进入文件夹

cd.. :返回上一个目录

mkdir :新建一个文件夹

rm :删除一个文件

ls :列出当前目录中的所有文件

mv :移动文件

reset :重置清屏

touch :新建一个文件

总结

Git Bash让我们了解原始 Git 方法的工作原理，使用命令行工具而不是 GUI 版本是有益的，尤其是当项目协作要求随着项目工作人数的增加而增加时。
